# E-Commerce Sales Analysis using Python

##  Project Overview
This project presents a comprehensive analysis of an e-commerce dataset to uncover actionable business insights. The goal is to transform raw transactional data into meaningful visualizations that help understand sales performance, customer behavior, and revenue trends.

By leveraging Python and powerful data analysis libraries, this project demonstrates the end-to-end workflow of a data analyst — from data cleaning to insight generation and visualization.

---

## Objectives
- Analyze overall sales performance
- Identify top-performing products
- Understand country-wise revenue distribution
- Evaluate customer payment preferences
- Discover monthly sales trends

---

## Tools & Technologies
- **Python**
- **Pandas** – Data Cleaning & Analysis  
- **Matplotlib** – Data Visualization  
- **Seaborn** – Advanced Visual Styling  

---

## Dataset Description
The dataset contains transactional data from an e-commerce platform, including:

- Order ID  
- Product Name  
- Quantity  
- Unit Price  
- Total Revenue  
- Order Date  
- Country  
- Payment Method  

---

## Data Analysis Workflow
1.  Data Cleaning
   - Converted date columns into datetime format  
   - Structured data for time-based analysis  

2.  Feature Engineering
   - Extracted monthly data from order dates  

3.   Exploratory Data Analysis (EDA)
   - Aggregated sales by month, product, country, and payment method  

4.   Data Visualization
   - Created multiple professional charts to highlight key insights  

---

##  Key Visualizations

## Monthly Sales Trend
- Line chart showing revenue growth over time  
- Helps identify seasonal patterns and trends  

## Top 10 Products by Revenue
- Horizontal bar chart highlighting best-performing products  

## Revenue by Country
- Comparison of sales contribution across different countries  

## Payment Method Distribution
- Donut chart showing customer payment preferences  

---

##  Key Insights
- Certain products consistently generate higher revenue  
- Sales trends reveal fluctuations across months  
- A few countries contribute the majority of revenue  
- Customers show clear preferences for specific payment methods
